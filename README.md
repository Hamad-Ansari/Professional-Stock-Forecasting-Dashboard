## 🚀 Overview

**Hammad Quant Pro** is a modern **AI-powered stock forecasting dashboard** built with **Streamlit**, **Python**, and **Machine Learning**.

It helps traders, investors, students, and analysts visualize stock market trends, generate future forecasts, and analyze performance using advanced forecasting models.

This project combines:

✅ Real-Time Market Data  
✅ Interactive Candlestick Charts  
✅ AI Price Forecasting  
✅ Buy / Sell Signals  
✅ Downloadable Forecast Reports  
✅ Professional Dark UI Dashboard

## 🎯 Key Features

### 📊 Market Dashboard
- Live Stock Price Data via Yahoo Finance
- Candlestick Chart Visualization
